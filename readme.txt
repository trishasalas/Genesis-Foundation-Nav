Stable tag: 0.1

Converts a multilevel dropdown menu to the Zurb Foundation menu which is responsive and mobile ready


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
== Frequently Asked Questions ==

Does it work with other themes than Genesis?

It should work with some modifications needed. For example, it works with Twenty Twelve if the wrapping markup si removed. Only leave the call to wp\_nav\_menu.
